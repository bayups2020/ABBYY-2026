## ❓ FAQ

**Is there a risk of detection or ban?**  
This tool operates locally on your system and does not interact with external servers for core functionality. There is a reduced risk with reasonable use, as it processes documents you provide. It is not an online cheat tool, so account bans are not a typical concern.

**I get an error "VCRUNTIME140.dll is missing" when launching the tool. How do I fix it?**  
This error indicates missing Visual C++ Redistributable files. The installer usually includes these, but if the issue persists, manually install the latest Microsoft Visual C++ Redistributable for Visual Studio 2015-2022 from the official Microsoft website (both x86 and x64).

**How do I clear temporary files or reset the tool?**  
The tool stores temporary files in the system's temp folder (%TEMP%\ABBYY_2026). You can safely delete all files in this location while the app is closed. To reset the settings, delete the configuration file located at %APPDATA%\ABBYY_2026\settings.json.
